    Added dark mode to my personal website. Also learned a bit about CSS media queries and how to have variables that differ based on dark mode: 

     :root {
     --background-color: #fff;
     --primary-color: #222;
}
 @media (prefers-color-scheme: dark) {
     :root {
         --background-color: #111;
         --primary-color: #ddd;
    }
}
